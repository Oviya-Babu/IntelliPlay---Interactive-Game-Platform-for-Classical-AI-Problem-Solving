# 🌟 Overview

Traditional methods of learning algorithms are often static and difficult to visualize. Students usually see only final outputs rather than understanding how algorithms think, search, optimize, and make decisions internally.

**IntelliPlay** bridges this gap by combining:

- 🧠 AI Agents
- 🎮 Gamified Learning
- ⚡ Real-Time Algorithm Visualization
- 📊 Step-by-Step Reasoning
- 🤖 Interactive AI Tutors
- 🎨 Modern Immersive UI

The platform enables users to learn Artificial Intelligence concepts interactively by playing games while simultaneously observing how the underlying algorithms solve problems in real time.

---

# 🎯 Problem Statement

Traditional algorithm education focuses heavily on theory and static outputs, making it difficult for students to understand the internal reasoning and decision-making process of AI algorithms.

Most learning platforms:

- Show only final answers
- Lack real-time interaction
- Do not visualize search spaces or constraints
- Fail to explain algorithmic thinking clearly

As a result, students struggle to:

- Understand algorithmic reasoning
- Visualize state-space exploration
- Learn heuristics and optimization
- Connect theory with practical execution

**IntelliPlay** solves this problem by transforming AI algorithms into interactive, explainable, and visual learning experiences.

---

# 💡 Project Objective

The objective of IntelliPlay is to create an AI-powered educational platform that:

- Makes algorithms interactive and engaging
- Visualizes algorithm execution step-by-step
- Explains AI decision-making in real time
- Helps users understand heuristics and constraints
- Converts theoretical AI concepts into practical learning experiences
- Encourages computational thinking through gameplay

# 🚀 Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/IntelliPlay.git
cd IntelliPlay
```

---

## 2️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on:

```bash
http://localhost:5173
```

---

## 3️⃣ Backend Setup

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

Backend runs on:

```bash
http://localhost:8000
```
